# RecognizeMe 🎓

An AI-powered smart attendance system that uses **face recognition** 
and **voice recognition** to automatically mark student attendance.

## Features
- 📸 Face recognition from classroom photos
- 🎙️ Voice-based student identification  
- 👨‍🏫 Teacher dashboard to manage subjects and view records
- 🎓 Student portal with face ID login
- 📊 Real-time attendance tracking via Supabase
- 📱 QR code sharing for class enrollment

## Tech Stack
- **Frontend** — Streamlit
- **Database** — Supabase (PostgreSQL)
- **Face AI** — dlib + face-recognition
- **Voice AI** — Resemblyzer
- **Language** — Python
